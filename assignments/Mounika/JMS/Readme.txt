Dependency(JMS Server) used in this example is ActiveMQ 
  ---> Need to install ActiveMQ in order to see the message.
  ---> Link to download ActiveMQ is http://activemq.apache.org/components/classic/download/
  ---> Run the activemq.bat(./bin/activemq.bat) file to start the JMS Server using the command activemq.bat start 
  ---> See the message in http://localhost:8161/admin/queues.jsp (after running code)
